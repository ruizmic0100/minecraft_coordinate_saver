# minecraft_coordinate_saver
Simple webpage that allows a user to save their minecraft coordinates and label them for later reference.

### Running

1. Clone the repository.
2. Install a web server of your choice. This example uses Nginx.
2.a. `sudo apt install nginx`
3. Open the firewall
3.a `sudo ufw allow 'Nginx HTTP'
4. `hostname -I` to show IP address of the machine.
4.a. `hostname` will also reveal the name of the machine.
5. If you want a friendlier link to the app. You can install `avahi-daemon` and access the server by typing in `http://[hostname].local`
